# VoltCue Mobile App

Futura app Android para Google Play.

## Objetivo

Convertir el control movil actual en una app real.

## Funciones Iniciales

- Login o vinculacion local.
- Agregar PC por PIN o QR.
- Ver estado de la PC.
- Apagar, reiniciar, suspender y bloquear.
- Historial.
- Demo gratis y Pro.
