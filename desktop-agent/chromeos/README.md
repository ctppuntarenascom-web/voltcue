# VoltCue ChromeOS / Chromebook

Objetivo: analizar soporte para Chromebook y ChromeOS.

## Viabilidad

ChromeOS es limitado para apagar el sistema desde apps externas. Por seguridad, Google no permite el mismo nivel de control que Windows, macOS o Linux.

## Enfoque Recomendado

Para Chromebook, VoltCue puede funcionar mejor como:

- App/controlador para apagar otras PCs.
- PWA o app Android instalada en ChromeOS.
- Panel para controlar computadoras Windows, macOS o Linux vinculadas.

## Limitaciones

- Apagar el Chromebook directamente puede no ser posible con permisos normales.
- Una extension de Chrome tampoco deberia tener permiso libre para apagar el sistema.
